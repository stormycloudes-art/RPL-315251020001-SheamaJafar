# Maintenance Plan

This maintenance plan outlines software maintenance activities for the Library Management System.

## Corrective Maintenance

1. Fix the login error when users enter valid credentials.
2. Correct incorrect book availability after a book is returned.
3. Resolve duplicate records appearing in the borrowing history.

## Adaptive Maintenance

1. Update the system to support the latest web browser versions.
2. Modify the database configuration to support a newer MySQL version.
3. Integrate with an updated email notification service.

## Perfective Maintenance

1. Add a book search feature with filters by title, author, and category.
2. Improve dashboard loading speed by optimizing database queries.

## Preventive Maintenance

1. Refactor the source code to improve readability and maintainability.
2. Update third-party libraries to their latest secure versions.
