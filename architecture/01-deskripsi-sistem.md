# System Description

## System Name
Online E-commerce System

## Purpose
The Online E-commerce System allows customers to browse products, make purchases, and track their orders online. It also enables administrators to manage products, inventory, customers, and orders efficiently.

## Primary Users
- Customer
- Administrator
## secondary Users
- Payment Gateway

## Main Features
- User Registration and Login
- Product Browsing
- Product Search
- Shopping Cart
- Checkout
- Online Payment
- Order Tracking
- Product Management
- Inventory Management
- Order Management
- Order Notification Management
