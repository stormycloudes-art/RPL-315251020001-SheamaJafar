# Physical View

The Online E-commerce System is deployed using the following components:

## Client
- Web browser used by customers and administrators.

## Web Server
- Hosts the application and handles client requests.

## Application Server
- Executes business logic such as authentication, shopping cart, orders, and payment processing.

## Database Server
- Stores user accounts, product information, orders, payments, and inventory data.

## Network

Users connect to the web server through the Internet. The web server communicates with the application server, which accesses the database server to retrieve and store information.
