# Physical View

The Online E-commerce System is deployed using the following components:

## Client
- Web browser used by customers and administrators.

## Web Server
- Hosts the application and handles client requests.

## Application Server
- Executes business logic such as authentication, product management, shopping cart management, order processing, payment processing, inventory management, and notification services.

## Database Server
- Stores user accounts, product information, shopping carts, orders, payment records, inventory data, and notification records.

## Network
Users connect to the web server through the Internet. The web server communicates with the application server, which accesses the database server to retrieve and store information. The application server also communicates securely with the external Payment Gateway to process online payments.
